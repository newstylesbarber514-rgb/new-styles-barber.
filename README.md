## 🚀 Como publicar

1. Envie o arquivo `New_Styles_Barber_Piva.html` para o repositório.  
2. Vá em **Settings → Pages → Source** e selecione `main / (root)`.  
3. Clique em **Save**.  
4. Seu site ficará disponível em poucos minutos no link acima.
